#### Hissa C — Block 3: فَائِدَہ ۲ (Non-mabni zarf rule) + 4th Quran ayah

Rule (PDF Block 3): *"ظُرُوْف غَیْر مَبْنِی جب جُملہ یا اِذْ کی طرف مضاف ہوں تو اُن کا فَتْحَہ پر مَبْنِی ہونا جائز ہے"*

Yaani: Mu'arrab (غیر مبنی) zarf — jab kisi complete jumla ya اِذْ ki taraf Mudaf ho → uska fatha par mabni hona jaiz (permissible).

Yeh rule kahaan se aata hai *(meri taraf se grammar context; book ne explicit framing nahi di)*: Normally Mu'arrab kalimah ka akhri harf Amil ke saath badalta (i'rab leta). Magar yeh specific zarf-class jab jumla/اِذْ ki taraf Mudaf ho, woh i'rab chhor kar fatha-mabni ban sakta hai — yeh exception rule hai.

Misaal — 4th Quran ayah is consecutive 4-page run mein:
- Book attribution: *"کَقَوْلِہٖ تَعَالٰی"* (Allah Ta'ala ke qol ke mutabiq) — preserve verbatim (Section 18 + 19 same pattern continuation)
- Ayah﴿هٰذَا یَوْمُ یَنْفَعُ الصَّادِقِیْنَ صِدْقُہُمْ﴾
- Book ka apna grammatical explanation: *"تو یہاں یوم کو مفتوح پڑھنا جائز ہے"* — yaani: "yahaan یَوْمُ (damma) ko یَوْمَ (fatha) padhna jaiz hai".
- Reason: یَوْمُ ke baad یَنْفَعُ الصَّادِقِیْنَ صِدْقُہُمْ = ek complete jumla hai. Yaani یَوْمُ Mudaf hai is jumla ki taraf — to فَائِدَہ ۲ ka rule apply hota.

⚠️ Book ne attribution "کَقَوْلِہٖ تَعَالٰی" explicitly diya — preserve karna lazmi. Surah Al-Ma'idah / 5:119 type memory plant NAHI karte.

2 nayi forms (Block 3 PDF):
- یَوْمَئِذٍ — "us din" (compound: یَوْمَ + اِذٍ = یَوْمَئِذٍ). Yeh ek shanaakhti form hai jo classical Arabic mein common hai.
- حِیْنَئِذٍ — "us waqt" (compound: حِیْنَ + اِذٍ = حِیْنَئِذٍ). Same pattern.

Compound formation *(meri taraf se observation; book ne formal rule nahi di)*: Dono forms ka pattern = base zarf (یَوْمَ/حِیْنَ) + اِذٍ (tanween-kasra waala اِذْ) = compound form. Book ne sirf misaalein dikhayi — formal Sarf-side rule (e.g., "yeh murakkab-mazji form hai" type detail) NAHI di. Hum bhi mat extrapolate.

Cross-context *(meri taraf se)*: یَوْمَئِذٍ Quran mein dozens of times aata (Surah-of-Reckoning context typically) — magar hum yahaan book ke beyond koi Quranic reference NAHI add karte.
